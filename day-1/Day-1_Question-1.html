Various methods in console function


console.log():

This method is used to output to the console,the string,array,number,code or anything inside the()will be displayed on the console

Example:

console.log("Hello there!"),this will be printed on the console as :

console.error():

his is used to log error messages to the console, it is useful in testing the code.

example:

console.error(' An error occured.') this displays a text in red saying An error occured.

console.warn():

This is used to log warning message to the console, it is displayed in yellow color.

example:

console.warn('Warning!!')

console.time() and console.timeEnd():

This is used to find the running time of a block of code, the console.time() is used to denote the start of the timer and console.timeEnd() to denote the end of the timer. The time elapsed is shown in milpseconds.

Example:

console.time(); for(i=0;i<1000;i++){
console.log("hi")
}
console.timeEnd();

console.table():

This method allows us to generate a table inside a console. The input must be an array or an object which will be shown as a table.

example:

console.table({'a':1, 'b':2})
